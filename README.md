## GENERATE QRCODE WITH PYTHON

### Requirements
- Python 3.8+
- pipenv

### Installation
```bash
$ python -m venv venv
$ venv/bin/activate

$ pip install -r requirements.txt
```

### Run
```bash
$ python app/main.py
```
